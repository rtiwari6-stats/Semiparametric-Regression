3. Run the mgcv fit to the data with the default number of knots (K=8) and with both K = 4 and K = 23 knots and using the cubic spline option as I have done. Save the model fit objects, e.g., gam4, gam8 and gam23

library(mgcv)
## Warning: package 'mgcv' was built under R version 4.2.3

gam4 = mgcv::gam(Strontium.Ratio~s(Age, k=6,bs="cr"), data=fossil)
gam8 = mgcv::gam(Strontium.Ratio~s(Age, k=10,bs="cr"), data=fossil)
gam23 = mgcv::gam(Strontium.Ratio~s(Age, k=25,bs="cr"), data=fossil)
